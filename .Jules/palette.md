## 2025-12-23 - Inconsistent Localization in Chat Components
**Learning:** The chat interface mixes English ("Send a message...") and Italian ("Trascina un file qui...", "Registra messaggio vocale"). This inconsistency creates cognitive load and confuses screen readers if the language attribute isn't switched.
**Action:** Future efforts should standardize on English (the primary UI language) or implement a proper i18n system to handle language switching consistently.
